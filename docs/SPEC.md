# 講義音声ノートAI（Lecture Audio Note AI）

## 1. 概要

大学の講義音声ファイルをアップロードすると、Gemini APIを利用して以下を自動生成するスマホ対応Webアプリを開発する。

### 出力内容

* 文字起こし（全文）
* 要約（3行程度）
* 要点（3〜5個の箇条書き）

本アプリは MVP（Minimum Viable Product）として実装し、まずは最小構成で動作することを優先する。

---

# 2. 開発方針

## 優先順位

# 7. Gemini処理仕様

## Step1

音声ファイルをGeminiへ送信

目的：

文字起こし生成

プロンプト：

あなたは高精度な音声文字起こしAIです。

音声内容を日本語で正確に文字起こししてください。

出力は文字起こし本文のみ返してください。

余計な説明は不要です。

---

## Step2

文字起こし結果をGeminiへ送信

目的：

要約・要点生成

プロンプト：

以下の講義内容を要約してください。

要件：

* 要約は3行程度
* 要点は3〜5個
* 日本語で出力

出力形式：

## 要約

（要約文）

## 要点

* 項目1
* 項目2
* 項目3

講義内容：

{{TRANSCRIPT}}

# 12. セキュリティ要件

必須

* APIキーをGitHubへコミットしない
* APIキーはlocalStorage保存のみ
* 個人情報を含む音声は利用しない

禁止

* ソースコードへのAPIキー直書き
* APIキーのログ出力

Lecture Audio Note AI — frontend MVP

概要
- 本リポジトリはフロントエンド単体の MVP です。講義音声をアップロードして文字起こし・要約・要点抽出を行うシンプルなデモを目指します。

ローカルでの実行（最小手順）
1. リポジトリをクローン / 移動
2. 静的サーバーで配信:
```bash
python3 -m http.server 8000
```
3. ブラウザで http://localhost:8000 を開き、画面の `APIキー設定` に Gemini のアクセストークン（Bearer トークン）を保存してから音声をアップロードし、「解析開始」を押してください。

重要な注意（CORS と認証）
- ブラウザから直接 Google の API にアクセスすると、CORS（プリフライト）でブロックされることがあります。これはブラウザのセキュリティ仕様によるもので、回避するにはサーバー側プロキシが必要です。
- このリポジトリはフロントエンド優先の MVP のため、サーバーは含めていません。サーバー経由での安全な運用を行う場合は別途プロキシを用意してください。

API キー（アクセストークン）
- 開発テスト用に `gcloud auth application-default print-access-token` で得たアクセストークンを `APIキー設定` に貼ると動作確認できます（短時間で期限切れになります）。本番ではサービスアカウントを用いたサーバー管理を推奨します。

GitHub で公開する際の注意
- `.env` やアクセストークンをリポジトリにコミットしないでください。公開リポジトリでは API キー管理に注意してください。
